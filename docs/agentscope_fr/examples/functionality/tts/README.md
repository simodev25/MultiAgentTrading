# TTS (Text-to-Speech) dans AgentScope

Cet exemple montre comment intégrer le modèle DashScope Realtime TTS avec `ReActAgent` pour permettre la sortie audio.
L'agent peut prononcer ses réponses en temps réel.

Cet exemple utilise le modèle DashScope Realtime TTS, mais vous pouvez également passer à d'autres modèles TTS pris en charge par AgentScope, par ex.
OpenAI, Gemini, etc.

Pour exécuter l'exemple, lancez :

```bash
python main.py
```
