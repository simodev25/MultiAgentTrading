# Exemple d'agent de recherche approfondie avec le modèle Qwen-Deep-Research

## Ce que cet exemple démontre

Cet exemple montre une implémentation d'Agent avec le modèle **Qwen-Deep-Research** utilisant le framework AgentScope. Il peut décomposer des problèmes complexes, utilise des recherches web pour effectuer des analyses et génère des rapports de recherche.

Référence : https://www.alibabacloud.com/help/en/model-studio/qwen-deep-research

## Prérequis

- Python 3.10 ou supérieur
- Clé API DashScope d'[Alibaba Cloud](https://dashscope.console.aliyun.com/)

## Comment exécuter cet exemple
1. **Définir la variable d'environnement** :
   ```bash
   export DASHSCOPE_API_KEY="your_dashscope_api_key_here"
   ```
2. **Exécuter le script** :
    ```bash
   python main.py
   ```
